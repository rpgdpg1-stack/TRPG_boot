# TRPG — гайд для Claude Code

Геймифицированный фитнес-трекер, Telegram Mini App. Стек: Vite + React 18 +
React Router 6 + Supabase + Telegram WebApp SDK. Деплой — Vercel. Стили — CSS (НЕ Tailwind).
Пользователь (Дмитрий) — нон-программист, общение на русском.

## Скилы проекта (`.claude/skills/`)

Подхватываются автоматически. **trpg-workflow** — входная точка для любого запроса по проекту;
из неё маршрутизация в остальные:

- **trpg-workflow** — рабочий процесс: как вести задачу, формат правок, коммиты+пуш,
  именование, грабли, добавление программ, обновление самих скилов.
- **trpg-supabase** — конвенции БД: RPC, RLS, миграции, Edge Functions.
- **trpg-ui** — дизайн-система: CSS-переменные, скругления, цвета, шрифты, press-эффекты,
  компоненты, анимации, хаптика Telegram.

## Чтение файлов — точечно

Под задачу читать только относящиеся файлы + их прямые связи, а не весь репозиторий.
Дерево проекта в trpg-workflow — карта для навигации, не повод читать всё.

## Коммиты + пуш

Дмитрий разрешил **авто-коммит и авто-push** в этом репо: после `npm run lint` + `npm run build`
(оба зелёные) коммит прямо в `main` (сообщение на русском, conventional commits) и сразу
`git push`. Ручное нажатие в GitHub Desktop не нужно. Проверка — на сайте GitHub; если что —
откат по просьбе.
