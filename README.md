# RPG Training App

Telegram Mini App для тренировок в стиле RPG.

## Технологии

- React 18 + Vite
- React Router (роутинг между страницами)
- Telegram Web App SDK (нативная интеграция с Телеграмом)
- Google Sheets как база данных упражнений
- Selectel CDN для медиа (картинки и видео упражнений)

## Команды

- `npm install` — установить зависимости
- `npm run dev` — запустить локально для разработки
- `npm run build` — собрать для продакшна

Деплой автоматический через Vercel при пуше в main.
